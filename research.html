Media Bias Analysis in Homelessness Coverage
Natural Language Processing & Large Language Models

Conducting research on media bias analysis using Large Language Models and Natural Language Processing to examine homelessness coverage across multiple cities, focusing on identifying patterns in media representation.

Research Objectives

  • Analyze biases in media coverage of homelessness across ten representative cities
  • Utilize LLMs and NLP with spaCy for comprehensive text analysis
  • Develop automated web scraping systems for data collection
  • Process and analyze data from ~70 news sources and social media platforms
  • Conduct large-scale analysis of media bias patterns

Methodology

Using Large Language Models and Natural Language Processing techniques with spaCy to analyze text data. Developing and deploying Python scripts using BeautifulSoup for efficient web scraping from multiple news sources. Processing data from cities including South Bend and San Francisco to identify regional differences in media coverage.

Current Progress

Successfully implemented web scraping pipeline for data collection from multiple sources. Developed NLP analysis framework using spaCy for bias detection. Currently processing and analyzing media coverage patterns across different cities and platforms.
